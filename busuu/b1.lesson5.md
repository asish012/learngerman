# Subordinate Conjunctions #

| English      | German  |
| ------------ | ------- |
| that         | dass    |
| if           | ob      |
| because      | weil    |
| when         | wenn    |
| when         | als     |
| since        | seit    |
| even though  | obwohl  |
| so that      | damit   |
| if / in case | falls   |
| while        | während |

* als : when two events happen at the same time, similar to "as"
* wenn: either expresses a condition ("if") or the moment something is taking place ("when")
* wann: asks for a specific time or date
